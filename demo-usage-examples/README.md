# Demo & Usage Examples Images

This directory contains screenshots and demo images for the README.md file.

## Required Images

Please add the following image files (PNG, JPG, or GIF format) to this directory:

### Discord Commands
- `daily-command-first.png` - First-time daily check-in command result
- `daily-command-streak.png` - Daily check-in with streak bonus (Day 5 example)
- `profile-command.png` - User profile command showing honor points, rank, and streak
- `leaderboard-command.png` - Private leaderboard command showing top 10 users
- `help-command.png` - Help command displaying all available commands
- `leaderboard-channel.png` - Live leaderboard embed in Discord channel

### Admin Dashboard
- `admin-dashboard-overview.png` - Full dashboard view with leaderboard table
- `admin-edit-modal.png` - Edit user modal for updating points and resetting streak
- `admin-dashboard-search.png` - Dashboard showing search functionality in action

### Terminal Outputs
- `terminal-bot-startup.png` - Bot initialization and startup logs
- `terminal-command-deployment.png` - Command deployment logs
- `terminal-message-points.png` - Message points logging output
- `terminal-leaderboard-update.png` - Leaderboard cron job update logs
- `terminal-bot-running.png` - Continuous bot running logs
- `terminal-error-handling.png` - Error handling and logging examples

### Backup System
- `backup-export.png` - Admin using `/backup export` command
- `backup-import.png` - Admin using `/backup import` command with file attachment

## Image Guidelines

- **Format:** PNG (preferred), JPG, or GIF
- **Size:** Recommended max width 1200px for screenshots
- **Quality:** Use high-quality screenshots that clearly show the UI/text
- **Naming:** Use exact filenames as listed above (case-sensitive)

## Adding Images

1. Take screenshots of your bot in action
2. Name the files exactly as listed above
3. Place them in this directory (`demo-usage-examples/`)
4. The images will automatically appear in the main README.md file
